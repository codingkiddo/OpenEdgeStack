# Security and Privacy

OpenEdgeStack must remain generic and vendor-neutral.

It must not include:

- customer data
- ISP/operator names
- proprietary telemetry schemas
- internal router logic
- proprietary firmware logic
- internal scoring algorithms
- private security detection rules
- production credentials
- real device identifiers

Use fake identifiers such as home-001, router-001, phone-001, malware-test.local, and DemoVendor.
